Basic theory behind the application

1 .This application is basically designed to make the user understand the difference and similarities between different arithmetic representation of integers.

2 .The key idea is to be able to observe and understand that some of the arithmetic representation behave isomorphically
hence any of them could be interchangebly used at the backend without the knowledge of the user.

3 .JavaScript has been used to provide an interctive experience for the user



How to use the application

Binary Addition-  Select any two rows that you want to add and press the ADD button. Note that the content of the table has been generated dynamically.

Results Displayed- The  result will be displayed in the results table.

OVERFLOW- If the resultant of the two numbers cannot be represented in given number of "bits" , then it is an overflow  which is indicated by the overflow flags.
                          There will be an alert generated along with a sound when an oveflow occurs .
                          ( The actual values requiring higher number of bits will be displayed in the result table for reference)

Refresh Page/Repeat- Press reset button for adding new set of numbers.
